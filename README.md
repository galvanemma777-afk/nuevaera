# Mega Luck Mobile Landing

Carpeta lista para subir a GitHub y publicar en Vercel.

## Cómo subir a GitHub

1. Creá un repositorio nuevo.
2. Subí todos los archivos de esta carpeta.
3. Entrá a Vercel.
4. Importá el repositorio.
5. Hacé Deploy.

## Cambiar WhatsApp

En `index.html`, buscá:

https://wa.me/0000000000

y reemplazalo por tu número con código de país.
